Google Sheets — Banco de Dados

Como configurar (faça uma vez só):
1. Crie uma planilha no Google Sheets
2. Vá em Extensões → Apps Script
3. Cole o script abaixo e clique em Implantar → Nova implantação
4. Tipo: App da Web · Executar como: Eu · Acesso: Qualquer pessoa
5. Copie a URL gerada e cole no campo abaixo · Clique em Salvar configuração

O sistema vai carregar automaticamente todos os servidores, modelos e tipos cadastrados na planilha.
Após salvar, clique em "Como fixar a URL" para nunca precisar reconfigurar em outro computador.
Como fixar a URL no arquivo (recomendado):
1. Abra o arquivo certidoes.html no Bloco de Notas ou VS Code
2. Encontre a linha: const SHEETS_URL_FIXA = 'https://script.google.com/macros/s/AKfycbwnToN2DImvoNgoImgxPHuxnedkyfjVcHwKmhNWs8RYbU3uIpTVlviEORn_Dff8jkhMyw/exec';
3. Substitua por: const SHEETS_URL_FIXA = 'SUA_URL_AQUI';
4. Salve o arquivo — agora funciona em qualquer computador sem reconfigurar!
